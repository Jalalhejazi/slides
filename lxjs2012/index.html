Lets talk web vs native.

Web technology can deliver a great user experience.

To say otherwise is a disservice to an important discipline, and tech.

Operating Systems are implemented, generally speaking, in C.

Web browsers are a user space program built on top of Operating Systems in, generally speaking, C code as well. (C++ moreso but stay with me here.)

There is no fundamental capability that is not available to the Operating System that cannot be exposed to a web browser.
